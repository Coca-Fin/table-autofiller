Скрипт для GoogleSheets, который упростил мне работу в компьютерном клубе по учету товара.
Выполняет построчные арифметические операции,
запоняет таблицу на основе уже известных ранее данных,
выводит разницу остатка и сумму в денежнеом эквиваленте.
# Пример входный данных
```
str ("
напитки
hqd энергетик 0.5л взрывная малина 15 190
сэндвичи
симба 8 195
")
```
